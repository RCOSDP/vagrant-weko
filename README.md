vagrant-weko
======

NIIが開発したリポジトリソフトウェア WEKO をインストールした環境を構築します。

WEKOを使った実験や開発を行う環境を、手軽に構築することを目的としています。

導入バージョン: weko 2.2.3

##必要ソフトウェア

- Virtualbox
 - https://www.virtualbox.org/wiki/Downloads
- Vagrant
 - http://www.vagrantup.com/downloads.html
- ansible
 - https://github.com/ansible/ansible


- ansible 2.2.0
